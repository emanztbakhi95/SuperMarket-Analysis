SuperMarket-Analysis
# Project Objective:
This project analyzes sales data to identify top-performing products across different cities, analyzed by month and customer gender.

# Key Analysis:
- City-Based Targeting: Identifying the most popular products per region.
- Temporal Trends: Analyzing sales fluctuations across different months.
- Demographic Insights: Understanding buying patterns based on Gender.

# Tools used: 
- Language: Python
- Libraries: Pandas / Matplotlib
- Methodology: Exploratory Data Analysis (EDA)
